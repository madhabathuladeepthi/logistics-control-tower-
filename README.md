# Logistics Control Tower (PySpark Project)

## Overview
This project analyzes shipment delays using PySpark.

## Features
- Delay calculation
- Aggregations (avg, max, min)
- Window functions (running avg, rolling avg)
- Export to Parquet

## Tech Stack
- PySpark
- Python
- SQLite (optional)
- Power BI

## Output
- Aggregated delay insights
- Route-level performance metrics

## Future Scope
- Scalable to handle 100x data
- Integration with real-time streaming
